# AURA 진행 계획

> 기본 Gaussian Splat 렌더링은 완료. 이제 SuperSplat 뷰어 수준으로 렌더링 퀄리티를 끌어올리는 것이 목표

---

### Task 1. 깊이 정렬(depth sorting) 적용하기
- 매 프레임 카메라 기준 z값 계산 → splat을 뒤→앞(back-to-front)으로 정렬 → 정렬 순서대로 그리기
- 완료 기준: 뿌연(hazy) 느낌이 사라지고 표면 대비가 또렷해짐

---

### Task 2. 안티앨리어싱(cov_2d 팽창) 처리하기
- `cov_2d` 대각에 작은 값(≈0.3) 더해 픽셀보다 작은 splat 안정화 → 깜빡임/사라짐 방지
- 완료 기준: 가시·가장자리 같은 미세 splat이 안정적으로 렌더됨

---

### Task 3. SH(시점 의존 색) 적용하기
- SH 데이터를 DataTexture로 GPU 전송 → vertex에서 `texelFetch`로 읽기 → 시점 방향 기반 SH 평가로 색 보정
- 완료 기준: 보는 각도에 따라 하이라이트/광택이 변하며 색 사실감이 올라감

---

### Task 4. 렌더 파이프라인 구조 분리하기
- `Viewer.tsx`에서 Three 렌더 로직을 `RenderEngine`으로 분리 → React는 얇은 래퍼, 계약은 `GaussianData` 타입
- 완료 기준: 렌더/웹 영역이 파일 단위로 분리되어 각각 독립 수정 가능

---

### Task 5. 렌더 루프 + OrbitControls 연결하기
- 단발 `renderer.render()` → `requestAnimationFrame` 루프로 전환 → `OrbitControls`로 카메라 회전/줌
- 완료 기준: 마우스로 모델을 회전·줌하며 실시간으로 확인 가능
