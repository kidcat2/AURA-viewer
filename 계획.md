# AURA 진행 계획

> PLY 파일을 읽어서 브라우저 화면에 Gaussian Point Cloud를 출력하는 것이 목표

---

### Task 1. Three.js Canvas를 React 화면에 출력하기
- `WebGLRenderer` 생성 → React `<div>`에 canvas 연결 → renderer 크기 설정
- 완료 기준: 브라우저에 검은 화면(빈 canvas) 출력 확인

---

### Task 2. Camera와 Scene을 연결하여 Point 출력하기
- 카메라 위치 설정 → `renderer.render(scene, camera)` 호출 → GaussianObject 씬에 추가
- 완료 기준: 브라우저 화면에 점(point) 출력 확인

---

### Task 3. Vertex / Fragment Shader 최소 구현하기
- `gl_Position`(점 위치), `gl_PointSize`(점 크기) 계산 → Fragment에서 단색 출력
- 완료 기준: 색이 있는 point cloud 출력 확인

---

### Task 4. PLY 색 데이터를 Shader로 연결하기
- geometry에 color attribute 등록 → vertex → fragment로 색상 전달 → 색상 반영
- 완료 기준: PLY 원본 색상이 화면에 출력됨

---

### Task 5. Point를 Gaussian Splat 형태로 렌더링하기
- 원형 falloff 적용 → alpha blending → soft edge 처리 → point size 조정
- 완료 기준: 부드러운 Gaussian 형태가 화면에 출력됨
